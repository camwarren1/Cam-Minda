What kind of computers you can use

Computers are offered in class, but you'll have homework where you need access to a computer outside of class. A computer that runs OSX or Windows would be easier to work with in this course.

Getting familiar with your computer:

The Terminal

You will want to try understand how to use the terminal in your computer. Try a few simple commands (open, list - "ls", change directory - "cd")

Setting your computer up:

Google Chrome

Download and install Google chrome. Chrome is the most adaptable and least buggy browser

Class Folder

Create a folder that you will use for class, most cenveniently on the desktop of your personal computer.

Preparing to code:

Git software

Visit git-scm to download and install Git software. This will allow your computer to update and share git folders/docs.

Atom

Visit Atom and download/install Atom (editor for HTML Docs).

Slack, Github

Make accounts at Github and Slack. Your Github account will allow you to store, download, and share Github files with yourself on different computers, or with others for Joint projects. Your slack account is a networking platform for developers.
